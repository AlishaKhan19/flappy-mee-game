# Flappy Mee
A web-based game inspired by the classic Flappy Bird game built using HTML, CSS, and JavaScript.

# About
Flappy Mee is a lightweight, browser-based obstacle avoidance game. The objective is to navigate through procedurally generated pipes to achieve the highest score possible. If you touch a pipe or the ground, the game ends.

# How to play
1. Go to  <a href="https://alishakhan19.github.io/flappy-mee-game/">live</a> site
2. Click the Play button on the screen.
3. Use the Space bar, Up arrow, or 'X' key to make your character jump.
4. Navigate through the pipes to increase your score.
5. If you crash, you can click the Replay button to try again.
